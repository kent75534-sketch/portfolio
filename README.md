# Cinematic Portfolio - Aditya Singh

A high-end, immersive portfolio website featuring interactive 3D rocky models, a cinematic HUD interface, and a custom boot sequence.

## 🚀 Deployment Guide

This is a static website and can be deployed in minutes to any major hosting platform.

### Option 1: Vercel (Recommended)
1. Go to [vercel.com](https://vercel.com).
2. Click **Add New** > **Project**.
3. Either connect your GitHub/GitLab/Bitbucket repository OR install the [Vercel CLI](https://vercel.com/cli).
4. If using the CLI:
   - Run `npm install -g vercel`
   - Run `vercel` in this directory.
5. Vercel will automatically detect the `index.html` and deploy it.

### Option 2: Netlify
1. Go to [netlify.com](https://netlify.com).
2. Log in and go to the **Sites** tab.
3. Simply **drag and drop** this folder into the "Drag and drop your site folder here" area.
4. Your site will be live instantly with a generated URL.

### Option 3: GitHub Pages
1. Create a new repository on GitHub.
2. Push this folder to the `main` branch.
3. Go to **Settings** > **Pages**.
4. Select the `main` branch as the source and click **Save**.

## 🛠️ Tech Stack
- **Three.js**: Interactive 3D rendering of rocky humanoid models.
- **GSAP & ScrollTrigger**: Cinematic transitions and floating animations.
- **Vanilla CSS**: Premium HUD design and glassmorphism.
- **HTML5**: Semantic structure and technical data integration.

## 📦 Asset Management
- **Models**: High-fidelity GLB assets are stored in the root directory.
- **Note**: Some models are large (~90MB). Ensure your hosting plan handles large assets well, or consider using a CDN for the models.
